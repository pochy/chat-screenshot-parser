# 🧹 クイックスタート: 古いリモートファイル削除

以前のテストで残っている Google Files API のファイルを削除する手順です。

---

## ⚡ 3ステップでクリーンアップ

### ステップ1: ファイル一覧を確認

```bash
python cleanup_remote_files.py --list
```

**出力例:**
```
================================================================================
ファイル名                                          作成日時
================================================================================
files/abc123xyz456                                 2026-01-13T10:30:00Z
files/def789uvw012                                 2026-01-13T11:45:00Z
files/ghi345mno678                                 2026-01-14T09:15:00Z
================================================================================
合計: 3件のファイル
```

---

### ステップ2: 古いファイルを削除

```bash
python cleanup_remote_files.py --delete-old --hours 24
```

**確認プロンプトが表示されます:**
```
24時間以上前のファイル: 2件
================================================================================
files/abc123xyz456                                 (36.5時間前)
files/def789uvw012                                 (30.2時間前)
================================================================================

2件のファイルを削除しますか? (yes/no): yes  ← 「yes」と入力
```

**削除完了:**
```
削除を開始します...

✅ 削除成功: files/abc123xyz456
✅ 削除成功: files/def789uvw012

================================================================================
削除完了: 2件成功, 0件失敗
================================================================================
```

---

### ステップ3: 削除を確認

```bash
python cleanup_remote_files.py --list
```

**出力例:**
```
================================================================================
ファイル名                                          作成日時
================================================================================
files/ghi345mno678                                 2026-01-14T09:15:00Z
================================================================================
合計: 1件のファイル
```

---

## 🚀 よく使うコマンド

### すべて削除（テストファイルを全部消す場合）

```bash
python cleanup_remote_files.py --delete-all
```

### 1時間以上前のファイルを削除

```bash
python cleanup_remote_files.py --delete-old --hours 1
```

### 48時間（2日）以上前のファイルを削除

```bash
python cleanup_remote_files.py --delete-old --hours 48
```

---

## ⚠️ エラーが出た場合

### エラー: Google API Key が必要です

環境変数を設定してください：

```bash
export GOOGLE_API_KEY=your-api-key-here
```

または .env ファイルに追加：

```bash
echo "GOOGLE_API_KEY=your-api-key-here" >> .env
```

### エラー: google-genai パッケージが必要です

```bash
pip install google-genai
```

## ✅ まとめ

1. `--list` で確認
2. `--delete-old` で古いファイルを削除
3. 再度 `--list` で確認

**注意:** 最新版の translate.py は自動的にファイルを削除するため、今後は手動クリーンアップが不要になります。
